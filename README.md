# Radar Logístico

Dashboard em Streamlit para monitoramento de atrasos em entregas criado para o desafio de dados da vivo 2026.

## Hospedagem

O arquivo `render.yaml` permite publicar o dashboard como Web Service no Render.

Aplicação publicada: https://dashboard-logistico-8sh7.onrender.com

## Recursos

- KPIs de atraso, prazo e criticidade;
- filtros por transportadora, região, status e prioridade;
- ranking de entregas problemáticas;
- comparação entre transportadoras;
- análise regional;
- tendência pela sequência operacional;
- upload opcional de arquivo CSV no mesmo formato da base;
- exportação dos dados filtrados.
